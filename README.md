# cordova-vueInit
cordova结合vue的初始化项目！一个vue打包成原生app的模板，项目中已经引用rem布局、vue-router，vuex，原生statusbar组件。
适合移动app开发！
## 项目运行
* 安装cordova
    > npm install -g cordova
* 克隆项目
    > git clione git@github.com:wuyi123321/cordova-vueInit.git
* 安装插件
    > 到项目目录下 npm install
* 安装平台
    > cordova platform add android --save
* 浏览器运行
    > npm run dev
* vue打包
    > npm run build
* 安卓打包
    > cordova build
